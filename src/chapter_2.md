# Chapter 2 : Creacion proyecto.

---
Primeros pasos en Rails

---

```ruby
gem install rails
rails new <application_name> -d postgresql
bundle install
bundle update
```

Con estos comandos creamos un proyecto en Rails con DB Postgre.

Ejecutamos el sevidor ...

```ruby
rails s
```
